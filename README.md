<h1 align="center"> Description </h1>
tTunes is a lightweight and easy-to-use music player for any device with a web browser installed.

tTunes can play music thanks to TurboWarp's TuneShark v3 extensions, so you can use any audio file with this software!

The software is in an early stage of development, so if you find any bugs, please let us know through this same channel on GitHub!

<h1 align="center"> Installation </h1>
To use tTunes, follow these steps:

1. Download the HTML file.

2. Double-click the downloaded HTML file or drag it to your preferred web browser.

3. Once the HTML file is ready, you can use tTunes as usual.

4. Enjoy your audio!
